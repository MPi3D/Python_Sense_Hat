# Python_Sense_Hat

Petits programmes en python pour le Sense Hat

[![Python Sense Hat](/python_sense_hat.png)](https://www.adafruit.com/product/2738)

## Install

``` sh
git clone https://github.com/MPi3D/Python_Sense_Hat.git
```

## Run

``` sh
cd Python_Sense_Hat
python3 Dice.py
python3 "Little Game.py"
python3 Point.py
```
